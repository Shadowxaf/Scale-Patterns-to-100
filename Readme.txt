--------------------------
  Scale Patterns to 100%
--------------------------

A Photoshop script to change all Patern Layer scales and Layer Style Pattern Overlay scales to 100%. Please note that Layer Style Pattern Overlays on Groups are not changed. Most code by David Jensen. Lots of assitance by Mike Hale, Paul MR and Jacobolus.

More discussion can be found on PS-Scripts:
http://www.ps-scripts.com/bb/viewtopic.php?f=9&t=4875&sid=0e6ab9c181714ad55fa3db6912cc1f54

Place in your /Adobe Photoshop CS6/Presets/Scripts/ folder and restart Photoshop for the script to appear in Scripts under the File menu.
